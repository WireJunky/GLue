LV.FloatArray

Overview:

 

The LV.FloatArray() command is used to create a Lua table that will be converted into a LabVIEW double floating point array by MLua when passed to a LabVIEW module or returned from a script. This type 'tagging' is necessary as Lua is a dynamically typed language with a single complex data type 'the Lua table'. With the additional meta information MLua can correctly convert the Lua table into a LabVIEW array. Two dimensional arrays are supported by inserting LV.FloatArray tables into a LV.FloatArray.

 

Note: Lua tables/arrays starting index is '1' not '0'

 

Usage:

 

The command can be used to either create a new empty array, or if supplied with a table to tag it as a float array.

 

1.Single dimension arrays:

 

 -- create a new empty single dimension float array

floatarray = LV.FloatArray()

floatarray[1] = 123.456

floatarray[2] = 456.123

return floatarray

 

 -- convert an existing table into a string array

array = {}

array[1] = 123.456

array[2] = 456.123

floatarray = LV.FloatArray(array)

return floatarray

 

 -- insert a table constructor as an argument to the function

floatarray = LV.FloatArray({123.456, 456.123})

return stringarray

 

2. Two dimension arrays:

 

 -- create a new empty two dimension float array

floatarray = LV.FloatArray()

floatarray[1] = LV.FloatArray()

return floatarray

 

 -- create a two dimensional array with two rows and two columns of data

floatarray = LV.FloatArray()

floatarray[1] = LV.FloatArray({123.456, 456.123})

floatarray[2] = LV.FloatArray({456.123, 123.456)

return floatarray
