Connector Pane


Close Instance

 Closes a previously opened instance of MLua freeing all associated resources.

Parameter(s)
-----------------

- MLua in: MLua reference to close.

- ms timeout (-1):The time to wait for the MLua reference to become available. A value of '-1' specifies an infinite timeout.

- error in: Describes error conditions that occur before this node runs. This VI runs normally even if an error occurred before this VI runs.

- error out: Contains error information. This output provides standard error out functionality.
